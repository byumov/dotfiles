# dotfiles

## Command line

### Fish

[fisher](https://github.com/jorgebucaran/fisher)

```bash
fisher install jorgebucaran/fisher
fisher install acomagu/fish-async-prompt
fisher install patrickf1/fzf.fish
fisher install danhper/fish-ssh-agent
```
